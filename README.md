# LunarClock

## これは何?
東方Project、十六夜咲夜のデスクトップマスコット時計です。時計だけでなくアプリケーションランチャーやファイル検索といった機能も実装している/する予定です。
 
## ご利用について
デスクトップに十六夜咲夜がいたらいいなぁと思っている十六夜咲夜ファン向けです。

## 動作環境
Microsoft Windows 10 以降を推奨します。
